Project Overview & Game Description

This mobile application is a fully functional, two-player version of the classic Tic-Tac-Toe game, developed using **MIT App Inventor**. The project was designed as a fun exercise in implementing core game logic and providing a visually distinct user experience.

The game features a high-contrast **red, white, and blue** American flag theme, replacing the traditional 'X' and 'O' with a turn-based system. The primary goal was to ensure the game accurately tracks the state of the 3x3 grid and immediately identifies a winner when three markers align horizontally, vertically, or diagonally.

Technical Skills Demonstrated

The development process allowed me to practice several core programming concepts:

  • **Event Handling:** Implementing block code that responds precisely to each of the nine button presses on the game board.
  • **Conditional Logic (If/Else):** Writing the complex logic required to check all eight winning scenarios.
  • **State Management:** Using variables (blocks) to manage player turns, track filled cells, and prevent double-clicks on the same square.
  • **Procedural Abstraction:** Creating separate functions (procedures) for tasks like checking the win condition and resetting the board to keep the code organized.
  • **Mobile UI Design:** Arranging game elements for optimal clarity and tapability on a mobile screen.
